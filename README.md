# CALIBRIX Attendance System

A digital attendance system built for the KEC Electronics & Instrumentation Engineering department hackathon.

## Features

- **Manual Attendance**: Search by roll number and mark participants present
- **Face Recognition**: Capture faces and use facial recognition to mark attendance
- **Hourly Tracking**: Track attendance across multiple hours
- **Export to Excel**: Download attendance reports
- **Admin Access**: Secure login for authorized personnel only

## Tech Stack

- React + TypeScript + Vite
- Supabase (Database & Storage)
- face-api.js (Face Detection & Recognition)
- Tailwind CSS + shadcn/ui

## Setup

1. Clone the repo
2. Run `npm install`
3. Set up your `.env` file with Supabase credentials
4. Run `npm run dev`
